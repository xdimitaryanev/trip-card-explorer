### Installation

1. **Download the project**
   ```bash
   git clone https://github.com/xdimitaryanev/trip-card-explorer.git
   cd trip-card-explorer
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the app**
   ```bash
   npm run dev
   ```
